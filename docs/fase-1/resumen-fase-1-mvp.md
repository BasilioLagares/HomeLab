# Proyecto HomeLab - Fase 1 MVP Local

## 1. Objetivo

Montar un primer laboratorio local en CachyOS usando Docker y Docker Compose, orientado a hostear proyectos propios, practicar bases de datos, monitorización y despliegues básicos.

Esta fase no busca crear un servidor definitivo ni exponer servicios a internet. El objetivo es validar que el HomeLab aporta valor usando el PC actual.
