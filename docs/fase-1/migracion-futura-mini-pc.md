# Proyecto HomeLab - Migración futura a mini PC

## 1. Objetivo

Este documento describe cómo se podría migrar el Proyecto HomeLab desde el PC actual con CachyOS a una máquina dedicada futura, como un mini PC con Debian Server.

La migración no se ejecuta todavía. El objetivo es dejar preparada una estrategia clara.
